# RS-31_Savchenko_Oleksii_RTOS_LR6

Номер групи: РС-31

Автор: Савченко Олексій

ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК Лабораторна робота 6. Широтно-імпульсна модуляція для управління силовими елементами на основі апаратних таймерів.
Обладнання: STM32F405, CubeIDE.

Мета: Ознайомитися з принципами роботи широтно-імпульсної модуляції (ШІМ) на STM32F405, навчитися використовувати апаратні таймери для керування силовими елементами та реалізувати програму з керуванням шириною імпульсу за допомогою кнопки.

Завдання до лабораторної роботи:

Варіант 1. Запрограмувати генерацію ШІМ на PD12 (TIM2) з частотою 1 кГц, керування шириною імпульсу кнопкою PA0.

Функціонал:

PWM Task: Генерує ШІМ сигнал з частотою 1 кГц на світлодіоді PE3 за допомогою таймера TIM2, за допомогою кнопки PC13 циклічно змінює duty cycle: 0% → 25% → 50% → 75% → 100% → 0%.

Підключення:

LED1: GND → LED → резистор 220 Ом → PA0

BUT1: GND → Button → резистор 220 Ом → PC13
